# Changelog

## [0.1.0](https://github.com/nraecher/nextcloud-news-digest/releases/tag/v0.1.0) (2026-07-19)

### Features

- Initial release of Nextcloud News Digest, a read-only tool that fetches unread news items from a Nextcloud News server and generates output documents (Markdown, PDF, plain text, JSON) for Open-Notebook import.
